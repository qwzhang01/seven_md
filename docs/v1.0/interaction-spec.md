# MD Mate 交互说明文档

> 版本：1.0 | 更新日期：2026-04-24  
> 本文档描述 MD Mate Markdown 编辑器的完整交互规范，适用于开发、测试和产品验收。

## 2. 标题栏交互

### 交通灯按钮
| 按钮 | 行为 | 快捷键 |
|------|------|--------|
| 🔴 关闭 | 关闭当前窗口 | Cmd+W |
| 🟡 最小化 | 最小化窗口到 Dock | Cmd+M |
| 🟢 最大化 | 切换全屏模式 | Ctrl+Cmd+F |

## 5. 活动栏与侧边栏交互

### 活动栏（Activity Bar）

活动栏位于侧边栏最左侧，是一列垂直排列的图标按钮，模仿 VS Code 的设计。

```
┌────┐
│ 📁 │  ← 资源管理器（默认选中）
├────┤
│ 🔍 │  ← 搜索
├────┤
│ 📑 │  ← 大纲
├────┤
│ 📋 │  ← 片段
└────┘
```

### 活动栏交互行为
| 行为 | 说明 |
|------|------|
| 点击图标 | 切换右侧内容面板并展开侧边栏 |
| 点击已激活的图标 | 收起侧边栏 |
| 图标激活指示 | 左侧显示竖线（accent 色） |
| 悬停效果 | 图标高亮、背景变亮 |
| 侧边栏已收起时点击 | 展开侧边栏并选中该图标对应面板 |
| 徽标提示 | 某些图标右上角可能显示小红点（如搜索结果数） |

### 侧边栏内容面板

侧边栏内容面板位于活动栏右侧，随活动栏图标切换而变化。

## 6. 编辑器交互

### 行号
- 显示在编辑区左侧
- 当前行号高亮
- 点击行号可选中整行

### 文本编辑
- 支持 contenteditable 直接编辑
- 支持 Tab 键插入缩进
- 支持 Shift+Tab 减少缩进
- 支持自动配对括号、引号
- 支持回车自动续行列表

### 语法高亮
编辑器通过 JS 动态为 Markdown 语法添加高亮：

| 语法 | 样式 |

| 加粗 | 粗体 + 强调色 |

| 行内代码 | 代码色 + 灰色背景 |
| 代码块 | 代码色显示 |
| 链接 | 链接色 + 下划线 |
| 引用 | 引用色 + 左边框 |
| 列表 | 列表色 |
| 水平线 | 水平线色 |
| 删除线 | 删除线 + 灰色 |
| 已完成任务 | 绿色 |
| 未完成任务 | 灰色 |

## 7. 预览区交互

### 预览渲染
- 编辑器内容变化时实时更新预览
- 预览支持完整的 Markdown 渲染（标题、列表、代码块、表格、图片等）
- 预览区域可独立滚动

### 预览区头部
- 左侧显示"预览"标签
- 右侧有"在新窗口打开"按钮

### 预览区右键菜单
- 与编辑器右键菜单相同

---

## 8. 分隔条（Gutter）交互

- 位于编辑器和预览之间
- 拖拽可调整编辑器与预览的宽度比例
- 悬停时光标变为 `↔`，分隔条高亮
- 拖拽中分隔条变为 accent 色

---

## 9. 视图模式切换

| 模式 | 说明 | 切换方式 |
|------|------|----------|
| 分栏视图 | 编辑器 + 预览并排 | 默认模式 |
| 仅编辑器 | 只显示编辑器 | 视图菜单 / 命令面板 / 工具栏按钮 |
| 仅预览 | 只显示预览 | 视图菜单 / 命令面板 / 工具栏按钮 |

切换视图模式时带有平滑过渡动画。

## 10. 查找替换栏交互

### 打开方式
| 操作 | 打开 | 快捷键 |
|------|------|--------|
| 查找 | 打开查找栏 | Ctrl+F |
| 替换 | 打开查找+替换栏 | Ctrl+H |

### 界面布局
```
┌─────────────────────────────────────────────────────┐
│ 🔍 [查找输入_____]  无结果  [▲] [▼]                 │
│ 🔁 [替换输入_____]  [替换] [全部替换]                │
│ ☐ Aa  ☐ W  ☐ .*                            [✕]     │
└─────────────────────────────────────────────────────┘
```

### 交互行为
| 行为 | 说明 |
|------|------|
| 输入查找词 | 实时高亮编辑器中所有匹配项 |
| 上/下箭头 | 跳转到上一个/下一个匹配 |
| 替换 | 替换当前匹配项并跳转到下一个 |
| 全部替换 | 替换所有匹配项 |
| Aa | 大小写敏感开关 |
| W | 全字匹配开关 |

## 20. 一致性规范

### 视觉一致性
- 所有可点击元素悬停时使用统一的 `--bg-hover` 背景
- 所有激活状态使用统一的 `--accent` 色
- 所有过渡动画使用统一的 `--transition-fast` / `--transition-normal` 时间
- 所有圆角使用统一的 `--radius-sm` / `--radius-md` / `--radius-lg`

### 交互一致性
- 相同操作在不同入口产生相同结果（如菜单、工具栏、右键菜单、命令面板）
- 所有弹出层支持 Esc 关闭
- 所有输入框支持 Enter 确认
- 所有面板切换带有过渡动画

### 反馈一致性
- 操作成功显示绿色 success 通知
- 操作失败显示红色 error 通知
- 需要注意显示黄色 warning 通知
- 一般信息显示蓝色 info 通知

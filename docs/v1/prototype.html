# MD Mate - AI 时代的 Markdown 编辑器 MD Mate 是为 AI 时代而生的 Markdown 编辑器，让你像写代码一样写文档。 ## 功能特性 ### 实时预览 编辑器右侧实时渲染 Markdown 预览，所见即所得。 ### 语法高亮 支持 Markdown 语法高亮，代码块支持多种语言着色。 ### 主题切换 支持多种主题切换，包括深色、浅色、Monokai、Solarized、Nord、Dracula、GitHub 等。 ### AI 辅助写作 集成 AI 助手，帮你智能编写与理解文档内容。 ## 安装与使用 ```bash # macOS 下载安装 brew install --cask md-mate # 或从官网下载 # https://mdmate.app ``` ## 开发计划 ### v1.0 里程碑 - [x] 基础 Markdown 编辑 - [x] 实时预览 - [x] 语法高亮 - [x] 主题切换 - [ ] AI 辅助写作 - [ ] 协同编辑 ### v2.0 规划 - [ ] 插件系统 - [ ] Git 集成 - [ ] 多语言支持 - [ ] 云同步 ## 贡献指南 欢迎贡献代码！请阅读 [贡献指南](CONTRIBUTING.md)。 ## 许可证 MIT License --- > 📌 **注意**: MD Mate 目前处于早期开发阶段，功能可能不完善。

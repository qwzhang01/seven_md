## 界面概览

```
┌─ 系统菜单栏（Tauri 原生：文件 / 编辑 / 视图 / 插入 / 格式 / 主题 / 帮助）┐
├─ 标题栏（交通灯 + 标签页 + 命令面板/侧边栏按钮）──────────────────────┤
├─ 工具栏（撤销重做 | H1 H2 H3 | 加粗斜体删除线 | ... | 引用 | AI）────┤
│ ┌──┐ ┌──────┐ ┌────────────┬────────────┐                               │
│ │活│ │侧边栏│ │   编辑器   │    预览    │                               │
│ │动│ │      │ │ (CodeMirror│  (Markdown │                               │
```

## 活动栏与侧边栏

### 活动栏

位于侧边栏最左侧，垂直排列 4 个图标：

| 图标 | 面板 | 功能 | 快捷键 |
|------|------|------|--------|
| 📁 | 资源管理器 | 文件树浏览（默认） | `Ctrl+Shift+E` |
| 🔍 | 搜索 | 全局搜索 | `Ctrl+Shift+F` |
| 📑 | 大纲 | 文档标题结构导航 | `Ctrl+Shift+O` |
| 📋 | 片段 | 常用 Markdown 模板 | — |

**交互**：点击图标切换面板并展开侧边栏，点击已激活图标收起侧边栏。激活状态左侧显示竖线标识。

### 资源管理器面板

- 上半部分显示**打开的文件**列表
- 下半部分显示**工作区文件树**
- 点击文件打开，点击文件夹展开/折叠
- 已修改文件显示蓝色圆点
- 悬停节标题时显示操作按钮（新建文件/新建文件夹/刷新/折叠全部）
- **右键菜单**：
  - 文件：打开 / 在新窗口打开 / 重命名 / 复制 / 剪切 / 删除 / 在资源管理器中显示 / 复制路径
  - 文件夹：新建文件 / 新建文件夹 / 重命名 / 复制 / 粘贴 / 删除 / 在资源管理器中显示

### 搜索面板

- 输入关键词实时搜索当前工作区所有文件
- 支持大小写敏感（Aa）、全字匹配（W）、正则表达式（.*）
- 搜索结果按文件分组，显示匹配行号和上下文
- 点击结果跳转到对应文件和行

### 大纲面板

- 自动解析当前文档的 H1-H6 标题层级
- **H1-H6 使用不同颜色区分层级**（便于快速定位）
- 支持筛选关键词过滤
- 点击标题跳转到编辑器对应位置
- 文档编辑时自动刷新

### 片段面板

分类展示常用模板（基础格式 / 高级模板 / Spec 模板），包括：

- Markdown 表格、代码块、任务列表、注释框
- 图片、脚注、折叠区块、Mermaid 流程图
- PRD 需求模板、API 文档模板、README 模板

点击片段项或右侧 ➕ 按钮在光标位置插入模板。

### 侧边栏尺寸调整

拖拽右边缘可调整宽度（180px ~ 500px）。

## 编辑器

基于 **CodeMirror 6** 的专业 Markdown 编辑器：

### 核心能力

- **行号显示**：左侧行号区，当前行高亮
- **语法高亮**：H1-H4 标题不同颜色和字号，加粗/斜体/代码/链接/引用/列表等均有语法着色
- **括号匹配**：自动高亮匹配的括号
- **自动配对**：输入括号、引号自动补全
- **列表续行**：在列表项末按 Enter 自动续行，空行退出
- **Tab 缩进**：Tab 插入缩进，Shift+Tab 减少缩进
- **查找替换**：`Ctrl+F` / `Ctrl+H`

### 右键菜单

在编辑器区域右键弹出菜单，包含：

- **剪切/复制/粘贴**
- **插入 ▶**（子菜单支持 **hover 自动展开**）：标题、加粗、斜体、代码、链接、图片、表格、列表、引用等
- **全选**
- **格式化文档**
- **AI 改写**

---

## 预览区

位于编辑器右侧，实时渲染 Markdown 内容：

- 支持 **GFM**（GitHub Flavored Markdown）：表格、任务列表、删除线
- 支持 **数学公式**（KaTeX 渲染）
- 支持 **代码语法高亮**（highlight.js）
- 预览区可独立滚动
- 头部右侧有"在新窗口打开"按钮

## 查找替换

| 操作 | 快捷键 |
|------|--------|
| 打开查找 | `Ctrl+F` |
| 打开查找+替换 | `Ctrl+H` |
| 查找下一个 | `Cmd+G` |
| 查找上一个 | `Cmd+Shift+G` |

功能：

- 输入查找词实时高亮所有匹配项
- 上/下箭头跳转匹配项
- 支持**替换**和**全部替换**
- 选项：大小写敏感（Aa）、全字匹配（W）、正则表达式（.*）
- `Enter` 下一个匹配，`Shift+Enter` 上一个，`Esc` 关闭

## 命令面板

**打开方式**：`Ctrl+Shift+P` / 视图菜单 / 标题栏按钮

- 输入关键词实时过滤命令列表
- 上下箭头移动选择，Enter 执行
- 支持的命令类别：主题切换、视图模式、编辑器操作、文件操作、插入操作、AI 功能

## AI 助手

按 `Ctrl+Shift+A` 或点击工具栏 AI 按钮打开侧边 AI 面板，提供 4 种模式：

| 模式 | 功能 |
|------|------|
| 💬 对话 | 与 AI 聊天讨论写作问题 |
| ✨ 改写 | 选中文本后选择风格（专业/随意/简洁/扩展）进行 AI 改写 |
| 🌐 翻译 | 选中文本后选择方向（中→英 / 英→中 / 中→日）进行翻译 |
| 💡 解释 | 选中代码或文本，AI 解释其含义 |

## 通知系统

屏幕右上角以 Toast 形式显示，共 4 种类型：**info / success / warning / error**。

### 交互行为

| 行为 | 说明 |
|------|------|
| 自动消失 | info/success 默认 3s，warning 5s，error 手动关闭 |
| **Hover 暂停** | 鼠标移入通知时暂停倒计时，移出后恢复 |
| 点击关闭 | 右上角 × 按钮立即关闭 |
| 堆叠展示 | 多个通知从上到下叠放，最多 5 条 |

## 多窗口

Seven Markdown 支持打开多个独立窗口：

- **快捷键**：`Ctrl+Shift+N`
- **菜单**：文件 → 新建窗口
- 每个窗口独立维护标签页、工作区、编辑器状态
- 窗口之间可独立切换工作区
- 适合同时处理多个项目或对比文档

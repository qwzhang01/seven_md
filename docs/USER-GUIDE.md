## 快速入门

### 系统要求

- macOS 10.15+ 或 Windows 10+ (64位)
- 约 50MB 磁盘空间

### 首次启动

1. 打开 Seven MD
2. 通过菜单栏 **文件 → 打开文件** 或 **打开文件夹** 开始
3. 也可以直接拖拽 .md 文件到窗口

## 文件操作

- **新建文件**：`Ctrl+N` 或 文件 → 新建文件
- **打开文件**：`Ctrl+O` 或 文件 → 打开文件
- **打开文件夹**：文件 → 打开文件夹
- **保存文件**：`Ctrl+S` 或 文件 → 保存
- **另存为**：`Ctrl+Shift+S`

## 编辑器

基于 CodeMirror 6 的专业编辑器：

- **语法高亮**：完整的 Markdown 语法着色
- **括号匹配**：自动高亮匹配的括号
- **自动配对**：输入括号/引号自动补全
- **列表续行**：在列表项末按 Enter 自动续行，空行退出
- **查找替换**：`Ctrl+F` 查找，`Ctrl+H` 查找替换
- **右键菜单**：剪切/复制/粘贴 + 插入子菜单 + AI 改写

## 预览

右侧实时预览区支持：

- GitHub Flavored Markdown（表格、任务列表、删除线等）
- 数学公式（KaTeX）
- 代码语法高亮
- 点击预览区头部的"在新窗口打开"按钮可打开独立预览窗口

## 侧边栏

通过活动栏图标切换 4 个面板：

| 图标 | 面板 | 功能 |
|------|------|------|
| 📁 | 资源管理器 | 文件树浏览 |
| 🔍 | 搜索 | 全局搜索 |
| 📑 | 大纲 | 文档结构导航 |
| 📋 | 代码片段 | 常用片段快速插入 |

## 主题切换

7 种内置主题：Light、Dark、Nord、Solarized Light、Solarized Dark、One Dark、Dracula

切换方式：
- 菜单栏 → 主题 → 选择主题
- 命令面板 → 输入 "theme"

## AI 助手

按 `Ctrl+Shift+M` 或点击工具栏 AI 按钮打开：

- **对话模式**：与 AI 聊天讨论 Markdown 写作
- **改写模式**：选中文字后使用 AI 改写

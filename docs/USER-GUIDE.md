## 标题栏与标签页

### 窗口控制（macOS 交通灯）

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| 🔴 关闭 | 关闭当前窗口 | `Cmd+W` |
| 🟡 最小化 | 最小化到 Dock | `Cmd+M` |
| 🟢 最大化 | 切换全屏 | `Ctrl+Cmd+F` |

### 标签页管理

- **点击标签**：切换到对应文件
- **关闭标签**：悬停后点击右侧 × 按钮
- **修改标志**：未保存的文件标签旁显示蓝色圆点
- **拖拽排序**：拖拽标签调整文件顺序

### 标题栏按钮

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| 命令面板 | 打开命令面板 | `Ctrl+Shift+P` |
| 侧边栏 | 显示/隐藏侧边栏 | `Ctrl+\`` |

## 菜单栏

菜单栏包含 7 个顶级菜单，点击展开下拉菜单，鼠标悬停可切换，菜单外点击关闭。

### 文件菜单

| 菜单项 | 功能 | 快捷键 |
|--------|------|--------|
| 新建文件 | 创建新 .md 文件 | `Ctrl+N` |
| 新建窗口 | 打开新窗口 | `Ctrl+Shift+N` |
| 打开文件 | 打开文件选择对话框 | `Ctrl+O` |
| 打开文件夹 | 打开文件夹 | — |
| 保存 | 保存当前文件 | `Ctrl+S` |
| 另存为 | 另存为新文件 | `Ctrl+Shift+S` |
| 导出为 PDF | 将文档导出为 PDF | — |
| 导出为 HTML | 将文档导出为 HTML | — |

### 编辑菜单

| 菜单项 | 功能 | 快捷键 |
|--------|------|--------|
| 撤销 / 重做 | 撤销/重做操作 | `Ctrl+Z` / `Ctrl+Shift+Z` |
| 剪切 / 复制 / 粘贴 | 文本操作 | `Ctrl+X` / `Ctrl+C` / `Ctrl+V` |
| 查找 | 打开查找栏 | `Ctrl+F` |
| 替换 | 打开查找替换栏 | `Ctrl+H` |

### 视图菜单

| 菜单项 | 功能 | 快捷键 |
|--------|------|--------|
| 命令面板 | 打开命令面板 | `Ctrl+Shift+P` |
| 切换侧边栏 | 显示/隐藏侧边栏 | `Ctrl+\`` |
| 放大 / 缩小 / 重置 | 调整字号 | `Ctrl+=` / `Ctrl+-` / `Ctrl+0` |
| 仅编辑器 / 仅预览 / 分栏视图 | 切换视图模式 | — |

### 插入菜单

标题、加粗、斜体、行内代码、代码块、链接、图片、表格、水平线、无序列表、有序列表、任务列表、引用。

### 格式菜单

加粗（`Ctrl+B`）、斜体（`Ctrl+I`）、删除线、标题 1-3、代码、链接。

### 主题菜单

提供 7 种主题可选：深色模式（默认）、浅色模式、Monokai、Solarized、Nord、Dracula、GitHub。点击立即切换，自动保存偏好。

### 帮助菜单

欢迎页、Markdown 指南、快捷键参考、关于、检查更新。

## 活动栏与侧边栏

### 活动栏

位于侧边栏最左侧，垂直排列 4 个图标：

| 图标 | 面板 | 功能 |
|------|------|------|
| 📁 | 资源管理器 | 文件树浏览（默认） |
| 🔍 | 搜索 | 全局搜索（支持正则、大小写、全字匹配） |
| 📑 | 大纲 | 文档标题结构导航 |
| 📋 | 片段 | 常用 Markdown 模板快速插入 |

**交互**：点击图标切换面板并展开侧边栏，点击已激活图标收起侧边栏。激活状态左侧显示竖线标识。

### 资源管理器面板

- 上半部分显示**打开的文件**列表
- 下半部分显示**工作区文件树**
- 点击文件打开，点击文件夹展开/折叠
- 已修改文件显示蓝色圆点
- 悬停节标题时显示操作按钮（新建文件/新建文件夹/刷新/折叠全部）

### 搜索面板

- 输入关键词实时搜索当前工作区所有文件
- 支持大小写敏感（Aa）、全字匹配（W）、正则表达式（.*）
- 搜索结果按文件分组，显示匹配行号和上下文
- 点击结果跳转到对应文件和行

### 大纲面板

- 自动解析当前文档的 H1-H6 标题层级
- 支持筛选关键词过滤
- 点击标题跳转到编辑器对应位置
- 文档编辑时自动刷新

### 片段面板

分类展示常用模板（基础格式 / 高级模板 / Spec 模板），包括：

- Markdown 表格、代码块、任务列表、注释框
- 图片、脚注、折叠区块、Mermaid 流程图
- PRD 需求模板、API 文档模板、README 模板

点击片段项或右侧 ➕ 按钮在光标位置插入模板。

### 侧边栏尺寸调整

拖拽右边缘可调整宽度（180px ~ 500px）。

## 编辑器

基于 **CodeMirror 6** 的专业 Markdown 编辑器：

### 核心能力

- **行号显示**：左侧行号区，当前行高亮
- **语法高亮**：H1-H4 标题不同颜色和字号，加粗/斜体/代码/链接/引用/列表等均有语法着色
- **括号匹配**：自动高亮匹配的括号
- **自动配对**：输入括号、引号自动补全
- **列表续行**：在列表项末按 Enter 自动续行，空行退出
- **Tab 缩进**：Tab 插入缩进，Shift+Tab 减少缩进
- **查找替换**：`Ctrl+F` / `Ctrl+H`

### 右键菜单

在编辑器区域右键弹出菜单，包含：

- **剪切/复制/粘贴**
- **插入子菜单**（标题、加粗、斜体、代码、链接、图片、表格、列表、引用等）
- **全选**
- **格式化文档**
- **AI 改写**

---

## 预览区

位于编辑器右侧，实时渲染 Markdown 内容：

- 支持 **GFM**（GitHub Flavored Markdown）：表格、任务列表、删除线
- 支持 **数学公式**（KaTeX 渲染）
- 支持 **代码语法高亮**（highlight.js）
- 预览区可独立滚动
- 头部右侧有"在新窗口打开"按钮

## 查找替换

| 操作 | 快捷键 |
|------|--------|
| 打开查找 | `Ctrl+F` |
| 打开查找+替换 | `Ctrl+H` |

功能：

- 输入查找词实时高亮所有匹配项
- 上/下箭头跳转匹配项
- 支持**替换**和**全部替换**
- 选项：大小写敏感（Aa）、全字匹配（W）、正则表达式（.*）
- `Enter` 下一个匹配，`Shift+Enter` 上一个，`Esc` 关闭

---

## 命令面板

**打开方式**：`Ctrl+Shift+P` / 视图菜单 / 标题栏按钮

- 输入关键词实时过滤命令列表
- 上下箭头移动选择，Enter 执行
- 支持的命令类别：主题切换、视图模式、编辑器操作、文件操作、插入操作、AI 功能

---

## AI 助手

按 `Ctrl+Shift+M` 或点击工具栏 AI 按钮打开侧边 AI 面板，提供 4 种模式：

| 模式 | 功能 |
|------|------|
| 💬 对话 | 与 AI 聊天讨论写作问题 |
| ✨ 改写 | 选中文本后选择风格（专业/随意/简洁/扩展）进行 AI 改写 |
| 🌐 翻译 | 选中文本后选择方向（中→英 / 英→中 / 中→日）进行翻译 |
| 💡 解释 | 选中代码或文本，AI 解释其含义 |

## 主题切换

Seven MD 内置 7 种主题：

| 主题 | 风格 |
|------|------|
| Dark（默认） | 经典深色 |
| Light | 经典浅色 |
| Monokai | Sublime 经典 |
| Solarized | 护眼暖色调 |
| Nord | 北极蓝调 |
| Dracula | 紫调暗色 |
| GitHub | GitHub 浅色 |

**切换方式**：
1. 菜单栏 → 主题 → 选择
2. 命令面板 → 输入"主题"

切换即时生效，全局覆盖所有 UI 元素，自动保存到本地。

| 区域 | 内容 | 点击行为 |
|------|------|----------|

| 模式 | 说明 | 切换方式 |
|------|------|----------|

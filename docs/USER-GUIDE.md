## 快速入门

### 系统要求

- macOS 10.15 (Catalina) 或更高版本
- 约 50MB 磁盘空间

### 首次启动

1. 打开 Seven MD 应用
2. 您将看到一个空白编辑器界面
3. 点击菜单栏的 **File → Open** 或使用快捷键 `Cmd+O` 打开 Markdown 文件

### 打开文件的三种方式

1. **菜单打开**：File → Open (或 `Cmd+O`)
2. **拖放打开**：将 `.md` 文件直接拖放到应用窗口
3. **打开文件夹**：File → Open Folder，然后在侧边栏中选择文件

```
┌─────────────────────────────────────────────────────────────┐
│  标题栏 (TitleBar)                                          │

│          │                        │                        │
│  侧边栏   │       编辑器           │        预览            │
│ (Sidebar) │    (EditorPane)       │    (PreviewPane)       │
│          │                        │                        │
```

### 标题栏

- **面包屑导航**：显示当前文件路径
- **窗口控制按钮**：最小化、最大化、关闭（遵循 macOS 设计规范）

- 显示打开文件夹的文件树结构
- 支持展开/折叠目录

### 编辑器

- 基于 CodeMirror 6 构建

- 实时同步编辑

### 预览面板

- 实时渲染 Markdown
- 支持代码高亮、数学公式、流程图等扩展语法

### 状态栏

- 显示当前文件信息
- 显示光标位置（行、列）
- 显示文件保存状态

## 文件操作

### 打开文件

**方式一：通过菜单**

1. 点击 File → Open
2. 在文件选择器中选择 `.md` 或 `.markdown` 文件
3. 点击"打开"

**方式二：拖放文件**

直接将 Markdown 文件拖放到应用窗口任意位置。

**方式三：打开文件夹**

1. 点击 File → Open Folder
2. 选择包含 Markdown 文件的文件夹
3. 在左侧侧边栏的文件树中点击文件打开

### 保存文件

- **保存**：File → Save (`Cmd+S`)
- **另存为**：File → Save As... (`Cmd+Shift+S`)

### 新建文件

1. 点击 File → New (`Cmd+N`)
2. 创建一个空白文档
3. 使用 Save As... 保存到指定位置

### 关闭文件

- File → Close (`Cmd+W`)

## 编辑器功能

### 基本编辑

- 支持标准文本编辑操作
- 自动缩进
- 自动配对括号和引号

### 语法高亮

编辑器支持完整的 Markdown 语法高亮：

- 标题 (h1-h6)
- 粗体、斜体、删除线
- 链接和图片
- 代码块和内联代码
- 列表（有序和无序）
- 引用块
- 表格
- 分隔线

### 实时同步

编辑器内容会实时同步到预览面板，无需手动刷新。

## 预览功能

### Markdown 扩展支持

Seven MD 支持标准 Markdown 以及以下扩展语法：

#### 代码块语法高亮

支持多种编程语言的语法高亮：

```javascript
function hello() {
  console.log("Hello, World!");
}
```

#### 数学公式 (KaTeX)

行内公式：$E = mc^2$

块级公式：

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

#### 流程图 (Mermaid)

```mermaid
graph LR
    A[开始] --> B[处理]
```

#### 表格

|------|--------|------|
| 打开 | Cmd+O | 打开文件 |
| 保存 | Cmd+S | 保存文件 |

#### 任务列表

- [x] 支持任务列表
- [ ] 待完成项
- [x] 已完成项

### 预览面板操作

- **滚动同步**：编辑器和预览面板滚动位置同步
- **点击跳转**：点击预览中的元素可跳转到对应源码位置

## 多标签页

Seven MD 支持同时打开多个文件，每个文件在独立的标签页中显示。

### 打开文件

- **新建标签页**：`Cmd+N` 创建空白标签页
- **打开文件**：`Cmd+O` 在新标签页中打开文件
- **从侧边栏打开**：点击文件树中的文件，在新标签页中打开
- **拖放打开**：将 Markdown 文件拖入应用窗口

### 标签页操作

- **切换标签页**：点击标签页标题，或使用 `Cmd+Tab` / `Cmd+Shift+Tab`
- **关闭标签页**：点击标签页上的 × 按钮，或使用 `Cmd+W`
- **重新排序**：拖动标签页标题可重新排列顺序
- **右键菜单**：右键点击标签页可访问更多操作（关闭其他、关闭右侧等）

### 未保存更改

- 有未保存更改的标签页会在标题旁显示 **●** 标记
- 关闭有未保存更改的标签页时，会弹出确认对话框
- 选择 **Save** 保存后关闭，**Don't Save** 直接关闭，**Cancel** 取消

### 标签页历史

- 关闭的标签页会保存在历史记录中（最多 10 个）
- 使用 `Cmd+Shift+T` 重新打开最近关闭的标签页
- File → Clear Tab History 可清除历史记录

### 标签页持久化

- 应用关闭时，所有标签页状态会自动保存
- 重新启动后，标签页会自动恢复
- 有未保存更改的标签页内容也会被保存

## 侧边栏

### 文件树导航

打开文件夹后，侧边栏会显示完整的目录结构：

- 📁 文件夹图标表示目录
- 📄 文件图标表示 Markdown 文件

### 侧边栏操作

- **展开/折叠目录**：点击文件夹图标
- **打开文件**：点击文件名

## 主题切换

### 支持的主题

- **浅色主题 (Light)**：适合白天使用
- **深色主题 (Dark)**：适合夜间使用
- **跟随系统**：自动跟随 macOS 系统设置

### 切换主题

**方式一：菜单切换**

View → Theme → 选择 Light / Dark / System

**方式二：快捷键**

View → Toggle Dark Mode (切换深色模式)

### 主题偏好保存

您的主题偏好会被保存，下次启动时自动应用。

## 常见问题

### Q: 为什么打开文件时没有反应？

**A:** 请确保：
1. 文件格式为 `.md` 或 `.markdown`
2. 文件未被其他程序锁定
3. 您有权限访问该文件

### Q: 预览面板显示不正确怎么办？

**A:** 尝试以下步骤：
1. 检查 Markdown 语法是否正确
2. 使用 View → Reload 重新加载
3. 如果问题持续，请提交 Issue

### Q: 如何更改默认保存位置？

**A:** 当前版本不支持更改默认位置，文件会保存到您选择的位置。

### Q: 支持 Windows 或 Linux 吗？

**A:** Seven MD 目前仅支持 macOS。未来可能会支持其他平台。

### Q: 如何报告问题或提出建议？

**A:** 请访问项目的 GitHub Issues 页面提交问题或建议。

### Q: 数学公式无法正确渲染？

**A:** 请确保：
1. 使用正确的 LaTeX 语法
2. 行内公式使用单个 `$` 符号
3. 块级公式使用双 `$$` 符号

### Q: 如何查看应用日志？

**A:** 开发者可以通过以下路径查看日志：
```
~/.seven-md/logs/
```

### Q: 应用崩溃后如何恢复？

**A:** Seven MD 具有自动错误恢复机制：
1. 应用会尝试自动恢复到崩溃前的状态
2. 如果文件有未保存的更改，可能会丢失
3. 建议养成经常保存的习惯 (`Cmd+S`)

## 一、已规划的功能

### v2.x 版本计划

#### 核心功能
- [ ] **Git 集成** ⭐ *当前重点*
  - 分支管理（查看、切换、创建分支）
  - 差异对比（编辑区与上次提交的差异高亮）
  - 提交历史（查看文件/项目的提交记录）
  - 提交操作（暂存、提交、推送）
  - GitHub/GitLab 远程仓库连接

- [ ] **OpenCode 集成** ⭐ *当前重点*
  - 将 Seven Markdown 打造成 **AI 写作研究 IDE**
  - 集成 OpenCode 作为后端 AI 引擎
  - 支持本地部署的 LLM（Llama、Qwen 等）
  - 写作助手功能：
    - 文献引用与管理
    - 研究报告模板
    - 自动摘要生成
    - 写作风格分析
    - 事实核查辅助
  - 支持 MCP (Model Context Protocol) 协议
  - 连接到各种 AI 服务（OpenAI、Claude、本地模型）

- [ ] **多窗口支持**
  - `Ctrl+Shift+N` 新建窗口
  - 窗口间独立状态
  - 窗口状态持久化

- [ ] **插件系统**
  - 插件加载机制
  - 插件市场/注册表
  - 基础 API（文件系统、编辑器扩展等）

- [ ] **自定义代码片段**
  - 用户自定义片段
  - 片段变量（日期、时间、光标位置）
  - 片段分类管理

- [ ] **同步协作编辑**
  - 多人同时编辑同一文档
  - 实时光标位置显示
  - 冲突解决机制

- [ ] **云端备份**
  - 自动备份到云存储
  - 版本历史
  - 一键恢复

#### 编辑增强
- [ ] **Vim/Emacs 键盘模式**
  - Normal/Insert/Visual 模式
  - 常用命令支持（dd, yy, p, /, etc.）
  - Vim 配置文件支持

- [ ] **LaTeX 实时预览**
  - 数学公式实时渲染
  - 公式编号和引用
  - 常见 LaTeX 语法支持

- [ ] **幻灯片模式**
  - Markdown 转幻灯片
  - 多种主题样式
  - 演讲者备注

- [ ] **Mermaid 图表编辑器**
  - 可视化图表编辑
  - 实时预览
  - 导出为图片

#### 高级功能
- [ ] **知识图谱**
  - 文档间链接关系可视化
  - 双向链接支持
  - 图谱导航

- [ ] **AI 写作助手增强**
  - 语法检查和纠错

  - 上下文感知的自动补全
  - 多语言翻译优化

## 十、贡献者指南

欢迎贡献代码！请参考 [CONTRIBUTING.md](./CONTRIBUTING.md)。

### 如何参与

1. **选择议题**：从上述列表选择一个感兴趣的功能
2. **讨论方案**：在 GitHub Issue 中讨论实现方案
3. **开始开发**：Fork 项目，创建分支，实现功能
4. **提交 PR**：附上测试和文档
5. **代码审查**：等待代码审查通过
6. **合并发布**：合并到 main 分支

### 新功能提案

新功能请先在 GitHub Discussions 中发起讨论，包括：
- 功能描述和用例
- 可能的实现方案
- 对现有功能的影响

# Windows 故障排除指南

## 概述

本文档提供 Seven MD 在 Windows 系统上可能遇到的常见问题及其解决方案。

## 安装问题

### 错误："应用程序无法启动"

**症状**: 双击应用图标后无响应或显示错误信息

**可能原因**:
- 缺少运行库
- 权限不足
- 系统兼容性问题

**解决方案**:
1. **安装 Visual C++ Redistributable**
   - 下载并安装 [Microsoft Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

2. **以管理员身份运行**
   - 右键点击应用图标 → "以管理员身份运行"

3. **检查系统兼容性**
   - 右键点击应用图标 → "属性" → "兼容性"
   - 尝试以兼容模式运行

### 错误："安装程序无法创建快捷方式"

**症状**: 安装过程中提示快捷方式创建失败

**解决方案**:
1. **关闭杀毒软件**
   - 临时禁用杀毒软件或防火墙
   - 将安装程序添加到白名单

2. **手动创建快捷方式**
   - 安装完成后，手动创建桌面快捷方式

### 错误："数字签名验证失败"

**症状**: Windows SmartScreen 阻止应用运行

**解决方案**:
1. **点击"更多信息"**
   - 在警告对话框中点击"更多信息"
   - 选择"仍要运行"

2. **禁用 SmartScreen（不推荐）**
   - 设置 → 应用和功能 → 应用执行别名
   - 关闭"检查应用和文件"

## 运行问题

### 问题：界面显示异常或模糊

**症状**: 文字模糊、界面元素错位

**解决方案**:
1. **调整 DPI 设置**
   - 右键点击应用图标 → "属性" → "兼容性"
   - 点击"更改高 DPI 设置"
   - 勾选"替代高 DPI 缩放行为"
   - 选择"系统"或"系统（增强）"

2. **更新显卡驱动**
   - 访问显卡制造商网站下载最新驱动

### 问题：应用启动缓慢

**症状**: 应用启动时间过长

**解决方案**:
1. **检查启动项**
   - 任务管理器 → 启动 → 禁用不必要的启动项

2. **清理临时文件**
   - 运行 `%temp%` 清理临时文件
   - 使用磁盘清理工具

3. **增加虚拟内存**
   - 系统属性 → 高级 → 性能设置
   - 高级 → 虚拟内存 → 更改

### 问题：文件关联无效

**症状**: 双击 .md 文件无法用 Seven MD 打开

**解决方案**:
1. **重新设置文件关联**
   - 右键点击 .md 文件 → "打开方式" → "选择其他应用"
   - 选择 Seven MD，勾选"始终使用此应用"

2. **使用应用内设置**
   - 打开 Seven MD
   - 设置 → 文件关联 → 重新关联 .md 文件

## 功能问题

### 问题：拖拽功能无效

**症状**: 无法通过拖拽打开文件

**解决方案**:
1. **检查用户账户控制**
   - 以管理员身份运行时可能限制拖拽功能
   - 尝试以普通用户身份运行

2. **重启 Windows Explorer**
   - 任务管理器 → 找到 Windows Explorer → 重启

### 问题：系统托盘图标不显示

**症状**: 最小化到系统托盘后图标不显示

**解决方案**:
1. **检查系统托盘设置**
   - 任务栏设置 → 选择哪些图标显示在任务栏上
   - 确保 Seven MD 图标设置为显示

2. **重启应用**
   - 完全退出应用后重新启动

### 问题：快捷键冲突

**症状**: 某些快捷键无法使用或触发其他应用

**解决方案**:
1. **检查全局快捷键**
   - 查看是否有其他应用使用了相同快捷键
   - 修改冲突的快捷键设置

2. **重置快捷键**
   - Seven MD 设置 → 快捷键 → 重置为默认

## 性能问题

### 问题：内存占用过高

**症状**: 应用运行缓慢，内存使用率持续升高

**解决方案**:
1. **限制打开文件数量**
   - 避免同时打开过多大型 Markdown 文件
   - 关闭不需要的文件标签

2. **清理缓存**
   - 设置 → 高级 → 清理缓存

3. **检查插件和扩展**
   - 禁用不必要的插件

### 问题：CPU 使用率异常

**症状**: 应用空闲时 CPU 使用率仍然很高

**解决方案**:
1. **检查后台进程**
   - 任务管理器 → 详细信息 → 查看 Seven MD 进程

2. **禁用实时预览**
   - 暂时关闭实时预览功能
   - 设置 → 编辑器 → 禁用实时预览

## 网络和更新问题

### 问题：无法检查更新

**症状**: 更新检查失败或超时

**解决方案**:
1. **检查网络连接**
   - 确保网络连接正常
   - 尝试切换网络环境

2. **配置代理设置**
   - 如有使用代理，配置正确的代理设置
   - 设置 → 网络 → 代理设置

3. **手动下载更新**
   - 从 GitHub Releases 页面手动下载最新版本

### 问题：自动更新失败

**症状**: 自动更新过程中出现错误

**解决方案**:
1. **关闭应用重试**
   - 完全退出应用后重新尝试更新

2. **手动安装更新**
   - 下载安装包进行手动覆盖安装

## 日志和调试

### 查看应用日志

日志文件位置：
```
%APPDATA%\Seven MD\logs\
C:\Users\[用户名]\AppData\Roaming\Seven MD\logs\
```

### 启用调试模式

1. **命令行启动调试**
   ```cmd
   "C:\Program Files\Seven MD\Seven MD.exe" --debug
   ```

2. **环境变量调试**
   - 设置环境变量 `SEVEN_MD_DEBUG=1`

### 生成错误报告

1. **使用内置报告工具**
   - 帮助 → 生成错误报告

2. **手动收集信息**
   - 应用版本号
   - Windows 版本
   - 错误截图
   - 日志文件内容

## 高级故障排除

### 重置应用设置

**完全重置**:
1. 关闭应用
2. 删除配置目录：`%APPDATA%\Seven MD\`
3. 重新启动应用

**部分重置**:
- 设置 → 高级 → 重置所有设置

### 检查系统完整性

```cmd
# 检查系统文件完整性
sfc /scannow

# 检查磁盘错误
chkdsk C: /f
```

### 性能监控

使用 Windows 性能监视器监控应用性能：
1. 运行 `perfmon`
2. 添加 Seven MD 进程的性能计数器
3. 监控内存、CPU、磁盘 I/O

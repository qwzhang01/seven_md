# Explorer Context Menu

## Overview

资源管理器中的文件和文件夹支持右键菜单操作。

## File Context Menu

| 菜单项 | 功能 |
|--------|------|
| 新建文件 | 在当前目录创建新 .md 文件 |
| 新建文件夹 | 在当前位置创建新文件夹 |
| 重命名 | 弹出输入框重命名 |
| 删除 | 删除文件（带确认） |
| 复制路径 | 复制绝对路径到剪贴板 |
| 在终端中打开 | 打开文件所在目录终端 |
| 在 Finder 中显示 | macOS 下定位文件 |

## Folder Context Menu

| 菜单项 | 功能 |
|--------|------|
| 新建文件 | 在当前目录创建新 .md 文件 |
| 新建文件夹 | 在当前目录创建新文件夹 |
| 重命名 | 弹出输入框重命名 |
| 删除 | 删除文件夹（带确认） |
| 复制路径 | 复制绝对路径 |
| 在终端中打开 | 打开文件夹终端 |
| 在 Finder 中显示 | macOS 下定位文件夹 |

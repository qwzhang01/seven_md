# Menu Dropdown Improvement

## Overview

完整的应用菜单栏，实现所有标准菜单项。

## Menu Structure

### 文件菜单
- 新建文件 (⌘N)
- 新建窗口 (⌘⇧N)
- 打开文件 (⌘O)
- 打开文件夹
- 保存 (⌘S)
- 另存为 (⌘⇧S)
- 导出为 PDF/HTML
- 关闭标签 (⌘W)
- 退出 (⌘Q)

### 编辑菜单
- 撤销/重做
- 剪切/复制/粘贴
- 全选
- 查找/替换

### 插入菜单
- 标题 (H1-H6)
- 格式（加粗/斜体/删除线）
- 代码（行内/代码块）
- 链接/图片
- 表格/水平线
- 列表（有序/无序/任务）
- 引用

### 视图菜单
- 命令面板
- 侧边栏/大纲面板
- 缩放
- 视图模式
- 全屏

### 主题菜单
- 7 个主题选项

### 窗口菜单
- 最小化/缩放
- 所有窗口置于前台

### 帮助菜单
- 欢迎页
- Markdown 指南
- 快捷键参考
- 关于
- 检查更新

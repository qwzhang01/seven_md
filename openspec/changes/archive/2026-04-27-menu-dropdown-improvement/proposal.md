## Why

应用菜单栏是用户与软件交互的核心入口。当前菜单栏缺少标准的下拉菜单，用户无法通过菜单进行文件操作、编辑格式化、视图切换等操作。需要参照主流 macOS 软件的菜单设计，实现完整的应用菜单。

### 技术实现

- 使用 Tauri 原生菜单 API (`@tauri-apps/api/menu`)
- 菜单配置集中管理（可考虑提取到配置文件）
- 菜单项事件统一通过 `listen` 监听器处理
- macOS 原生风格菜单（系统自动处理样式）

## Capabilities

### New Capabilities

- `app-menu-bar`: 完整的应用菜单栏

### Modified Capabilities

- `file-operations`: 文件菜单相关操作
- `edit-operations`: 编辑菜单相关操作
- `view-operations`: 视图菜单相关操作

## Impact

- 修改文件：`src-tauri/src/menu.rs` 或相关菜单配置
- 修改文件：`src-tauri/src/lib.rs` 中的菜单构建逻辑
- 修改文件：`src/AppV2.tsx` 中的事件监听器
- 需要新增菜单事件处理

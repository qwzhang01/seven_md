## Why

MD Mate 需要从现有的基础 Markdown 阅读器升级为**工业级 AI 驱动的 Markdown 编辑器**。当前版本功能有限，缺乏完整的编辑器体验（工具栏、活动栏、命令面板、AI 助手等）。基于最新的交互设计规范（`interaction-spec.md`），我们需要实现一套类似 VS Code 的专业编辑器界面，提供实时预览、多主题切换、AI 辅助写作、完整的文件管理等核心能力，打造「为 MD 而生，像写代码一样写文档」的极致体验。

## What Changes

本次变更将 MD Mate 从 **Markdown 阅读器** 全面升级为 **专业 Markdown 编辑器**，主要变更包括：

- **全新标题栏系统**：macOS 交通灯按钮 + 多标签页管理（支持拖拽排序、修改指示点）+ 命令面板/侧边栏快捷按钮
- **完整菜单栏**：文件 / 编辑 / 视图 / 插入 / 格式 / 主题 / 帮助 七大菜单，覆盖全部操作入口
- **专业工具栏**：撤销重做、标题层级（H1-H3）、文本格式化（加粗/斜体/删除线）、代码块、链接图片、列表、引用、表格、AI 助手按钮
- **VS Code 风格活动栏 + 侧边栏**：
  - 活动栏：资源管理器 / 搜索 / 大纲 / 片段 四个图标面板
  - 资源管理器：文件树展示、文件夹展开折叠、修改状态指示
  - 搜索面板：实时搜索、大小写敏感/全字匹配/正则选项
  - 大纲面板：文档标题结构导航
  - 片段面板：常用 Markdown 模板快速插入
- **增强编辑器核心**：行号显示、Markdown 语法高亮、自动配对括号引号、列表续行、右键上下文菜单
- **实时预览区**：与编辑器分栏显示，支持独立滚动、新窗口打开
- **命令面板（Command Palette）**：Ctrl+Shift+P 快速搜索执行全部命令
- **AI 助手面板**：对话 / 改写 / 翻译 / 解释 四种模式
- **主题系统**：7 种内置主题（Dark/Light/Monokai/Solarized/Nord/Dracula/GitHub），全局切换带过渡动画
- **状态栏**：Git 分支、同步状态、错误警告、光标位置、编码、换行符、语言模式
- **通知系统**：4 种类型通知（info/warning/error/success），右下角弹出带动画
- **查找替换栏**：编辑器内查找替换，支持正则/大小写/全字匹配选项
- **模态对话框**：确认对话框通用组件（未保存确认、删除确认等）
- **响应式适配**：桌面端完整布局 + 移动端自适应布局
- **键盘导航**：完整的 Tab/Enter/Esc/箭头 键盘支持

## Capabilities

### New Capabilities

以下为本次新增的功能模块，每个模块将对应创建独立的 spec 文件：

- `titlebar-system`: 标题栏系统 - macOS 交通灯、标签页管理（切换/拖拽/关闭/修改指示）、操作按钮（命令面板、侧边栏切换）
- `menubar-system`: 菜单栏系统 - 七大菜单（文件/编辑/视图/插入/格式/主题/帮助）完整交互逻辑和快捷键绑定
- `toolbar-system`: 工具栏系统 - 分组按钮布局（撤销重做/标题/文本格式/代码/链接图片/列表/其他/AI）、悬停 Tooltip、选中/非选中行为
- `activity-bar`: 活动栏 - 垂直图标栏（资源管理器/搜索/大纲/片段）、激活指示、悬停效果、徽标提示
- `sidebar-explorer`: 资源管理器面板 - 文件树渲染、展开折叠、文件打开、修改指示、活动文件高亮、操作按钮组
- `sidebar-search`: 搜索面板 - 实时搜索、搜索选项（Aa/W/.*）、结果列表、点击跳转、匹配高亮和计数
- `sidebar-outline`: 大纲面板 - 标题层级解析（H1-H4）、颜色标识、点击跳转、自动更新
- `sidebar-snippets`: 片段面板 - 预定义模板列表、预览文本、一键插入到光标位置
- `editor-core`: 编辑器核心 - CodeMirror 增强（行号、语法高亮、自动配对、列表续行）、右键上下文菜单
- `preview-pane`: 预览区 - 实时 Markdown 渲染、独立滚动、头部操作栏、右键菜单
- `command-palette`: 命令面板 - 模糊搜索、命令分类、键盘导航、快捷执行
- `ai-assistant`: AI 助手面板 - 四模式（对话/改写/翻译/解释）、消息输入输出、选中文本处理
- `theme-system`: 主题系统 - 7 种主题定义、CSS 变量切换、平滑过渡、持久化存储
- `status-bar-enhanced`: 增强状态栏 - 多区域信息展示（Git/同步/问题/光标/编码/语言）、点击交互
- `notification-system`: 通知系统 - 4 种类型、弹出动画、自动关闭（5s）、悬停暂停、堆叠显示
- `find-replace`: 查找替换栏 - 内联面板、实时高亮、上下一项、替换/全部替换、选项开关
- `modal-dialog`: 模态对话框 - 通用对话框组件、遮罩层、动画、键盘支持（Esc/Enter/Tab）
- `view-layout`: 视图布局管理 - 分栏/仅编辑器/仅预览 三种模式、分隔条拖拽调整、平滑过渡动画
- `keyboard-navigation`: 键盘导航系统 - 全局快捷键绑定、焦点管理、一致性交互
- `responsive-layout`: 响应式布局 - 桌面端（≥769px）完整布局、移动端（<768px）自适应

### Modified Capabilities

以下现有 capabilities 的需求发生变化，需要更新 spec：

- `tab-bar-ui`: 标签页需要增强拖拽排序、修改指示点蓝色圆点、关闭按钮悬停显示等交互
- `tab-management`: 标签管理需要支持更复杂的状态同步和持久化
- `file-operations`: 文件操作需要扩展（导出 PDF/HTML、新建窗口等）
- `editor-enhancements**: 编辑器增强需要整合到新的 editor-core 能力中
- `status-bar`: 状态栏需要大幅增强功能和交互

## Impact

### 影响范围

**前端代码 (React/TypeScript)**:
- `src/components/` - 几乎所有组件需要重构或新建（预计 40+ 组件）
- `src/hooks/` - 新增多自定义 hooks（编辑器状态、主题、命令面板等）
- `src/context/` - 新增/重构 React Context（编辑器、主题、AI、通知等）
- `src/styles/` - 主题 CSS 变量系统重构
- `src/types/` - TypeScript 类型定义扩展
- `src/utils/` - 工具函数库扩展（Markdown 解析、搜索、命令注册等）
- `src/i18n/` - 国际化文案大量新增
- `src/reducers/` - 状态管理重构（如使用）

**后端代码 (Tauri/Rust)**:
- `src-tauri/src/` - 可能需要新增 Tauri 命令（文件系统操作、AI 接口调用、原生对话框等）
- `src-tauri/tauri.conf.json` - 配置更新（窗口属性、菜单、权限等）

**依赖变更**:
- 可能新增依赖：AI SDK 客户端、更多 UI 图标库、状态管理库（如 zustand）
- 现有依赖升级：确保 CodeMirror 6 相关包版本兼容

**测试影响**:
- `src/test/` - 单元测试需要全面覆盖新组件
- `e2e/` - E2E 测试用例需覆盖核心交互流程

**构建产物**:
- 应用体积会增加（新功能、新依赖）
- 首次加载时间可能增加（需关注性能优化）
